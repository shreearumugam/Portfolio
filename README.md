## 🚀 Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/shreearumugam/Portfolio.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Portfolio
   ```

3. Open `index.html` in your web browser to view the portfolio.
